Template helper el

handlebars.el

Outputs html elements

{{#el}}content{{/el}}
{{{el content}}}
{{{el content=content}}}

Content is determined in the following way, stopping when it has a value

  1. Captured content If options.fn exists then the search stops here, even if the resulting value is empty
  2. Single positional params
  3. Multiple positional params Concatenated and assigned to content as an array
  4. content param
  5. el-content param
  6. contents param when passed as a string or array
  7. fallback param
